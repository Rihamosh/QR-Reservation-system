# QR-Reservation-system
Students attendance, class management and hall reservation system 

U-Attend is a system that consists of a web application and hybrid mobile application that facilitates attendance checking and saves time for both the students and the tutors as opposed to the traditional pen and paper system. This report discusses how the system relies on QR codes to facilitate the attendance taking process. The tutors will display the code from the web application to the students in the hall. Meanwhile, the students will access their mobile applications to scan the code and get their attendance confirmed.
